# Orchestration: UI Expert Workflow

Comprehensive UI analysis and improvement recommendations.

## Flow

The UI Expert Workflow follows a 4-step process:

1. **Trigger**: User invokes a UI-related skill or slash command
2. **Routing**: Claude matches the intent to UI analysis triggers
3. **Execution**: The ui-expert agent runs to analyze and provide recommendations
4. **Handoff**: Final output provides comprehensive UI insights and suggestions

## Agents

### ui-expert
A UI/UX specialist focused on frontend development, design systems, and user interface best practices.

**Skills**: component-analysis, design-system-audit

## Usage

Use this workflow in Claude Code or Cursor when you need:
- Component analysis and optimization
- Design system audits and recommendations
- UI/UX best practices guidance
- Frontend development insights

Invoke through skills like `component-analysis` or `design-system-audit` to get expert UI feedback and actionable improvement suggestions.
