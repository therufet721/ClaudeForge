# Claude Agent Registry

## Workflow: UI Expert Workflow

Comprehensive UI analysis and improvement recommendations.

## Agents

### ui-expert
- A UI/UX specialist focused on frontend development, design systems, and user interface best practices.
- Skills: component-analysis, design-system-audit

## Skills

- **component-analysis** (ui-expert): Analyzes UI components for structure, styling, and best practices. Use when user asks to review components, check UI code, or analyze frontend architecture.
- **design-system-audit** (ui-expert): Audits design system consistency and usage. Use when user wants to check design tokens, component consistency, or design system compliance.
